# Paws with Hope Animal Centre

## Project Overview

Paws with Hope Animal Centre is a small nonprofit project focused on rescuing, rehabilitating, and rehoming abandoned or abused animals. This repository contains the website files for the organisation: a simple, responsive HTML/CSS site with pages for adoption listings, donations, contact, and general information.

## Current Site Status

- Static site built with plain HTML, CSS, and a small amount of JavaScript.
- Pages included in this workspace: [index.html](index.html), [about.html](about.html), [adopt.html](adopt.html), [donate.html](donate.html), [contact.html](contact.html).

## Goals and Objectives

- Increase successful pet adoptions by providing clear listings and application information.
- Collect online donations and volunteer registrations via clear calls to action.
- Provide a clean, image-focused, and mobile-responsive experience.
- Improve discoverability and user trust through organized content.

## Key Features

- Homepage with featured animals and main CTAs.
- Adopt a Pet page with images, descriptions, and application details.
- Donations page with instructions and links to secure donation methods.
- Contact page with form and contact details.
- Volunteer sign-up flow (simple form on the site).

## Design & UX

- Color palette: green (care/nature), white (clean), brown (warmth).
- Typography: bold sans-serif for headings (Arial/Roboto), clean readable body fonts.
- Mobile-first and image-forward layout for animal listings.

## How to View Locally

Open `index.html` in your browser. For a simple local server (recommended for testing relative paths), run:

```bash
python -m http.server 8000
# then open http://localhost:8000 in your browser
```
